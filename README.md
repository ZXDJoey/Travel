# travel App

> A Vue.js travel web app

## 技术栈

> vue2 + vuex + vue-router + aiosx + webpack + es6/7

## 项目运行

``` bash
# git clone
git clone https://github.com/ZXDJoey/Travel.git

# cd project
cd Travel

# project init
npm init

# project run(localhost:8080)
npm run dev
```
